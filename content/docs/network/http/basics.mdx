## Specifications

HTTP是由这些[IETF](https://www.ietf.org/) RFC和IANA注册中心定义的。

**Core Specifications**

- RFC 9110: [HTTP Semantics](https://httpwg.org/specs/rfc9110.html)
- RFC 9111: [HTTP Caching](https://httpwg.org/specs/rfc9111.html)

- RFC 9112: [HTTP/1.1](https://httpwg.org/specs/rfc9112.html)
- RFC 9113: [HTTP/2](https://httpwg.org/specs/rfc9113.html)
- RFC 9114: [HTTP/3](https://httpwg.org/specs/rfc9114.html)

更高版本的HTTP提供字段压缩：
- RFC 7541: [HPACK Header Compression for HTTP/2](https://httpwg.org/specs/rfc7541.html)
- RFC 9204: [QPACK Field Compression for HTTP/3](https://httpwg.org/specs/rfc9204.html)

这些RFC共同取代了所有先前定义HTTP的RFC，包括**RFC 1945**、**RFC 2068**、**RFC 2616**、**RFC 2617**、**RFC 7230-5**和**RFC 7540**。

[更多HTTP规范文档](https://httpwg.org/specs/)
